1. Start MongoDB (if running locally)

   source venv/bin/activate
   uvicorn server:app --host 0.0.0.0 --port 8001 --reload

3. Start Frontend (Terminal 2):
   cd frontend
   yarn start  # or npm start

4. Open http://localhost:3000 in your browser
